# 🚦 Smart Traffic Management System using Computer Vision

## 📌 Project Overview
The Smart Traffic Management System is a Python-based application that uses Computer Vision to detect and monitor traffic in real time. The system analyzes traffic from video input and helps estimate traffic density, making it useful for intelligent traffic signal management and monitoring.

---

## ✨ Features
- 🚗 Real-time vehicle detection
- 🎥 Video-based traffic monitoring
- 🖥️ User-friendly graphical interface
- 📊 Traffic density analysis
- ⚡ Automated traffic sensing using AI techniques

---

## 🛠️ Technologies Used

- Python
- OpenCV
- PyTorch
- NumPy
- Tkinter (GUI)

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Traffic-Management-System.git
```

### 2. Open the project folder

```bash
cd Traffic-Management-System
```

### 3. Install required libraries

```bash
pip install opencv-python torch numpy pillow
```

### 4. Run the application

```bash
python interface.py
```

or

```bash
python main.py
```
